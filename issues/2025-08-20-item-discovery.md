# Item Discovery System

**Date**: 2025-08-20  
**Status**: In Progress - Phase 1 Complete  
**Priority**: Medium  
**Category**: Feature  

## Description

Implement an item discovery system that places AI-generated and predefined items throughout the world, with treasure chests, hidden caches, enemy drops, and contextually appropriate loot for each region.

## Details

**What is the requirement?**
Create an item discovery system with the following features:

- **AI-Generated Items**: Context-appropriate items for each region type
- **Treasure Containers**: Chests, caches, and hidden compartments
- **Enemy Loot Drops**: Items from defeated enemies
- **Environmental Items**: Items found by searching rooms thoroughly
- **Item Rarity System**: Common to legendary item quality levels
- **Regional Appropriateness**: Items fitting the region's theme and style
- **Hidden Item Discovery**: Search mechanics to find concealed items

**Acceptance criteria:**
- [ ] AI generation of items appropriate to region themes
- [ ] Treasure containers spawned in appropriate rooms
- [ ] `search` command to find hidden items in rooms
- [ ] Loot drops from defeated enemies
- [ ] Item rarity system affecting spawn rates and properties
- [ ] Regional item themes (mansion antiques, forest herbs, cave gems)
- [ ] Item descriptions generated by AI with atmospheric detail
- [ ] Configurable item spawn rates and rarity distribution

### Treasure Container System
```sql
CREATE TABLE treasure_containers (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  room_id INTEGER NOT NULL,
  container_type TEXT NOT NULL, -- chest, cache, altar, bookshelf
  name TEXT NOT NULL,
  description TEXT NOT NULL,
  is_hidden BOOLEAN DEFAULT FALSE,
  is_opened BOOLEAN DEFAULT FALSE,
  required_item TEXT, -- key or tool needed to open
  search_difficulty INTEGER DEFAULT 10, -- DC for finding hidden containers
  loot_table TEXT, -- JSON array of possible items
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (room_id) REFERENCES rooms(id) ON DELETE CASCADE
);
```

### Loot Drop System
```typescript
const generateEnemyLoot = (enemy: Enemy, playerLevel: number): Item[] => {
  const loot: Item[] = [];
  const dropChance = enemy.loot_drop_chance || 0.5;
  
  if (Math.random() < dropChance) {
    // Generate appropriate loot for enemy type and region
    const rarity = rollItemRarity(enemy.level);
    const item = generateRegionalItem(enemy.region, rarity);
    loot.push(item);
  }
  
  // Always drop some coins
  const coinAmount = enemy.level * (10 + Math.random() * 20);
  loot.push(createCoinItem(coinAmount));
  
  return loot;
};
```
