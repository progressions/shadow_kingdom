# Ink TUI Implementation - Lite Summary

Implement a modern three-section terminal interface using Ink (React for terminals) to replace the current readline-based interface in Shadow Kingdom. The new TUI features a scrollable game pane, bordered input bar, and compact status pane to enhance the text adventure gameplay experience with clear visual separation and command history navigation.

## Key Points
- Replace readline interface with Ink-based three-section TUI
- Add scrollable game pane with command history navigation
- Create bordered input bar and compact status display
