# Product Mission

> Last Updated: 2025-08-25

## Pitch

Shadow Kingdom is an AI-powered text adventure game that revolutionizes procedural world generation through region-based storytelling. Unlike traditional text adventures with static worlds, Shadow Kingdom creates infinite, thematically coherent fantasy realms where every room, character, and story element is dynamically generated by AI while maintaining spatial consistency and narrative cohesion.

The game combines the nostalgia of classic text adventures with cutting-edge AI technology, creating a unique experience where players explore a living world that expands intelligently around them. Each playthrough offers a completely different kingdom to explore, with AI-generated regions, characters, and stories that respond to player choices and create emergent narratives.

## The Problem

### Current Text Adventure Limitations
1. **Static Worlds**: Traditional text adventures feature fixed, hand-crafted worlds that become predictable after completion, offering little replay value
2. **Limited Scope**: Manual content creation restricts world size and complexity, leading to relatively small game worlds compared to modern standards
3. **Outdated Interfaces**: Most text adventures still use primitive command-line interfaces that haven't evolved with modern terminal UI capabilities

### AI-Generated Content Challenges  
1. **Inconsistency Issues**: Existing AI-generated games often produce contradictory or nonsensical content due to lack of spatial and narrative consistency mechanisms
2. **Poor Context Awareness**: AI systems typically lack sufficient context about game state, player history, and world rules to generate coherent responses
3. **Generic Output**: AI-generated content often feels bland or repetitive without sophisticated prompt engineering and context management

### Modern Gaming Expectations
1. **Interface Standards**: Players expect polished, responsive interfaces with command history, scrolling, and visual feedback
2. **Persistent Progress**: Modern games need save systems, session management, and progress tracking
3. **Natural Language**: Players want to interact using natural language, not memorize obscure command syntax

## Differentiators

### Revolutionary World Generation
- **Region-Based Architecture**: Unique 12-room region generation system that creates thematically coherent areas (mansions, forests, caves, towns) while maintaining perfect spatial consistency
- **Distance-Based Probability**: Sophisticated algorithm that determines region transitions based on distance, creating natural geographic flow (15% base probability + 12% per distance unit)
- **Visit-to-Lock Mechanism**: Innovative spatial consistency system that prevents phantom connections and maintains world integrity as players explore

### Advanced AI Integration
- **Claude API Integration**: Transition from Grok to Claude API for superior natural language processing and context understanding
- **Contextual Generation**: AI receives comprehensive room context, regional themes, and player history to generate coherent, relevant content
- **Fallback Systems**: Robust fallback mechanisms ensure gameplay continuity even during AI service outages

### Modern Terminal Interface
- **Ink-Based TUI**: React-inspired terminal user interface with scrollable content, floating input bars, and dynamic status areas
- **Command History**: Persistent command history with arrow key navigation and session persistence
- **Smart Command Parsing**: AI-powered command disambiguation that understands natural language input and player intent

### Intelligent Target Resolution
- **Universal Target System**: Centralized target disambiguation service that handles "all" commands (pickup all, drop all, give all to merchant) with consistent logic across all game commands
- **Context-Aware Matching**: Smart matching system that understands game context and resolves ambiguous targets based on player location and available entities

## Key Features

### Core Gameplay Features
- **Procedural World Generation**: Infinite kingdom exploration with AI-generated rooms, characters, and items that maintain thematic coherence within regions
- **Natural Language Interaction**: Advanced command parsing that understands natural language input, synonyms, and player intent without requiring exact command syntax
- **Dynamic Character Generation**: AI-created NPCs and enemies that appear contextually appropriate to their environment with unique personalities and backstories
- **Intelligent Item Systems**: Context-aware item placement and interaction with items that have purpose and meaning within their generated environments

### Technical Innovation Features
- **Region-Based Expansion**: World grows in thematic 12-room regions (Castle Keep, Dark Forest, Crystal Caves, Mountain Village) maintaining geographical and narrative consistency
- **Background Generation Service**: Proactive world expansion that generates new areas ahead of player exploration without blocking gameplay
- **Connection-Based Architecture**: Sophisticated spatial system that pre-creates directional connections that AI fills, eliminating phantom exits and maintaining perfect world coherence
- **Session Management**: Robust save/load system with programmatic interfaces for testing and automation

### User Experience Features  
- **Modern Terminal UI**: Professional interface with scrollable content areas, status bars, and command history that rivals modern terminal applications
- **Command History**: Persistent command storage with arrow key navigation that survives game sessions
- **Smart Error Handling**: Graceful degradation when AI services are unavailable, with fallback content and clear user feedback
- **Comprehensive Logging**: Development-friendly logging systems with configurable verbosity for debugging and development

### Developer Experience Features
- **Prisma ORM Integration**: Modern database layer with type-safe queries and automated migrations
- **Comprehensive Testing**: 800+ automated tests covering all major systems with isolated test environments  
- **Programmatic Interface**: Command-line interface for automated testing and development workflows
