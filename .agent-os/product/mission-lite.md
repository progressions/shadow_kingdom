# Shadow Kingdom Mission (Lite)

Shadow Kingdom is an AI-powered text adventure game that creates infinite, thematically coherent fantasy worlds through region-based procedural generation. Unlike traditional static text adventures, every room, character, and story element is dynamically generated while maintaining perfect spatial consistency.

Our key differentiator is the region-based world generation system that creates 12-room thematic areas (mansions, forests, caves, towns) with sophisticated distance probability mechanics and visit-to-lock spatial consistency. Combined with Claude API integration for superior natural language processing and a modern Ink-based terminal interface, Shadow Kingdom delivers the nostalgia of classic text adventures enhanced with cutting-edge AI technology.
